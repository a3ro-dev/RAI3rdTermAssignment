# ICSE Syllabus Questions

## Overview
A collection of practice problems and data handling tasks tailored for the ICSE syllabus. Includes various formats like CSV, JSON, and Python scripts.

## Contents
- **Scripts (q1-q6)**: Specific solutions to syllabus-based coding questions.
- **[data.csv](data.csv)** & **[data.json](data.json)**: Sample datasets for parsing practice.
- **[syllabus_practice.py](syllabus_practice.py)**: Solutions for common ICSE programming problems.
- **[turtle_heart.py](turtle_heart.py)**: Graphics programming exercise using the Turtle module.
- **[syllabus_asset.jpg](syllabus_asset.jpg)**: Supporting image asset for syllabus tasks.
